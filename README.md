QR Code Scanner

How it works : There is a txt file where you can add your own URL(It may be some website link or app link) and then run it.
Used npm packages QR Image for generating images, inquirer for taking url's

Steps on how You can use it 
step1: clone Source code
step2: Install node modules which are required
step3 : insert your url in the URL.txt file and save it
step4 : run the js file
